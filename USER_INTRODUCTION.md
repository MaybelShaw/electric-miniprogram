# 家电商城 - 用户使用指南

## 平台简介

家电商城是一个智能家电购物平台，对接海尔等品牌厂商，提供正品家电和配送安装服务。

**核心功能：**
- 商品浏览与搜索
- 购物车管理
- 在线下单支付
- 订单跟踪
- 物流查询

---

## 主要功能

### 1. 商品浏览
- 首页推荐商品
- 分类浏览（冰箱、洗衣机、空调等）
- 搜索商品（支持按名称、品牌、价格筛选）
- 查看商品详情（图片、参数、价格）

### 2. 购物下单
- 加入购物车
- 选择收货地址
- 提交订单
- 在线支付（微信支付）

### 3. 订单管理
- 查看订单状态（待支付/待发货/待收货/已完成）
- 查询物流信息
- 取消订单
- 申请退款

### 4. 个人中心
- 微信一键登录
- 管理收货地址
- 查看收藏商品



-# 使用流程

### 购买流程
1. 打开小程序（微信/支付宝）
2. 浏览或搜索商品
3. 加入购物车或立即购买
4. 填写收货地址
5. 提交订单并支付
6. 查看订单状态和物流

### 地址管理
1. 进入"我的" → "收货地址"
2. 点击"新增地址"
3. 填写联系人、电话、详细地址
4. 可设置为默认地址
